⚠️ Important

Le numéro dédié — ce qu'il faut absolument savoir

C'est le point le plus important à comprendre avant de commencer.

⚠️

Vous avez besoin d'un numéro WhatsApp séparé pour le bot

WaziBot doit être connecté à un numéro WhatsApp dédié à votre boutique. Ce numéro ne peut plus être utilisé comme WhatsApp classique une fois connecté. Vous ne pourrez plus envoyer des messages à vos amis ou famille depuis ce numéro.

❌ Utiliser votre numéro personnel

  • Vous perdez accès à WhatsApp classique
  • Vos groupes familiaux disparaissent
  • Vos contacts personnels ne peuvent plus vous joindre
  • Mélange entre vie privée et boutique
  • Le bot répond à tous vos messages

✅ Utiliser un numéro dédié boutique

  • Votre numéro personnel reste intact
  • Séparation claire vie privée / boutique
  • Image professionnelle pour vos clients
  • Le bot gère uniquement les clients
  • Coût : ~500 FCFA/mois (puce MTN/Moov)
💡

Conseil pratique

Achetez une nouvelle puce MTN, Moov ou Togocel dédiée à votre boutique. Donnez ce numéro à vos clients pour les commandes WhatsApp. Gardez votre numéro personnel pour votre vie privée. C'est la configuration idéale et la plus simple.

Démarrage

Comment démarrer en 5 étapes

Suivez ces étapes dans l'ordre pour lancer votre boutique WaziBot.

1

Obtenez une puce dédiée pour votre boutique

Achetez une nouvelle puce MTN, Moov, Togocel ou autre opérateur. Ce sera le numéro WhatsApp de votre boutique que vous donnerez à vos clients.

💡

Coût estimé

~500 FCFA pour la puce + abonnement opérateur mensuel. Un investissement minime comparé au temps gagné !

2

Créez votre compte WaziBot

Remplissez le formulaire d'inscription sur notre site avec les informations de votre boutique. Ajoutez vos premiers produits directement dans le formulaire.

Vous recevrez un ID unique — conservez-le précieusement, c'est votre clé d'accès au dashboard.

3

Votre site boutique est créé automatiquement

Dès votre inscription, WaziBot crée automatiquement un mini-site professionnel pour votre boutique avec tous vos produits.

Vous avez déjà un site ?

Pas de problème ! Dans votre dashboard → Paramètres → Site web, vous pouvez indiquer l'URL de votre propre site. Les clients y seront redirigés.

4

Partagez votre lien et commencez à vendre

Votre boutique est prête. Partagez le lien de votre site et votre numéro WhatsApp avec vos clients !

4a

Copiez le lien depuis Dashboard → Paramètres → Site web et partagez-le sur WhatsApp, Instagram, Facebook.

4b
Gérez vos commandes depuis le dashboard

Accédez à votre dashboard directement depuis le bas de votre site boutique → lien "Espace commerçant". Ou allez sur /merchant avec votre ID boutique.

Questions fréquentes

Vous avez des questions ?

Voici les réponses aux questions les plus posées.

Que se passe-t-il si un client m'envoie un message en dehors des heures de travail ?▼
Le bot répond automatiquement 24h/24, 7j/7 — même la nuit, les weekends et les jours fériés. Vos clients ne seront jamais sans réponse.
Est-ce que je peux modifier mes produits après l'inscription ?▼
Oui, depuis votre dashboard vous pouvez ajouter, modifier ou supprimer des produits à tout moment. Les changements sont pris en compte immédiatement par le bot.
Puis-je utiliser mon numéro personnel pour le bot ?▼
Techniquement oui, mais ce n'est pas recommandé. Une fois connecté à WaziBot, ce numéro ne fonctionnera plus comme WhatsApp classique. Nous vous conseillons fortement d'utiliser une puce dédiée à votre boutique.
Comment le bot sait-il quoi répondre ?▼
Le bot utilise Claude, une intelligence artificielle d'Anthropic. Il connaît votre catalogue, votre description de boutique et votre personnalité. Plus vous remplissez votre profil avec de détails, meilleures seront les réponses.
Que se passe-t-il quand mon essai gratuit expire ?▼
Vous recevrez un message WhatsApp 3 jours avant l'expiration, puis la veille. Le jour de l'expiration, votre dashboard vous proposera de choisir un plan. Le bot sera suspendu jusqu'au renouvellement de votre abonnement via My Touchpoint.
Mon ID de boutique est-il sécurisé ?▼
Votre ID est un identifiant unique de 36 caractères aléatoires — il est pratiquement impossible à deviner. Conservez-le précieusement et ne le partagez avec personne.
Comment fonctionne le bot WhatsApp ?▼
Le bot répond automatiquement aux messages WhatsApp de vos clients 24h/24. Il présente votre catalogue, prend les commandes et vous notifie à chaque nouvelle vente. Pas de configuration technique requise !
