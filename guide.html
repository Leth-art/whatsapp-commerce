Comment rendre votre boutique visible

Suivez ces étapes pour que vos clients trouvent et commandent sur votre site

01

Créez votre boutique

Inscrivez-vous sur WaziBot, choisissez votre secteur d'activité (mode, alimentation, tech...) et votre boutique est créée automatiquement avec un lien unique.

⏱️ Moins de 2 minutes pour démarrer
02

Ajoutez vos produits avec photos

Depuis votre dashboard → Produits, ajoutez chaque article avec une photo nette, un nom clair et un prix. Un bon visuel = 3x plus de ventes.

📸 Conseil : photographiez vos produits en plein jour, sur fond blanc ou neutre
03

Copiez votre lien boutique

Allez dans Paramètres → Site web et copiez votre lien unique. C'est ce lien que vos clients utilisent pour visiter et commander sur votre boutique.

wazibot.com/boutique/votre-nom
04

Partagez partout — chaque jour

👥
Groupes WhatsApp

📘
Groupes Facebook
Postez dans "Achat/vente Lomé", groupes de votre ville. Gratuit et efficace.
📸
Instagram / TikTok
Mettez le lien en bio. Publiez des photos de produits avec votre lien boutique.

05

Le bot répond automatiquement 24h/24

Quand un client visite votre boutique, un assistant chat apparaît. Il répond aux questions sur vos produits, vos prix et votre livraison — même pendant que vous dormez.

🤖 Vous n'avez rien à configurer, le bot est activé automatiquement
06

Recevez les commandes sur WhatsApp

Quand un client confirme une commande sur votre boutique, vous recevez immédiatement une notification WhatsApp avec tous ses détails. Gérez vos commandes depuis votre dashboard.

📱 Notification instantanée à chaque nouvelle commande

Questions fréquentes

Vous avez des questions ?

Voici les réponses aux questions les plus posées.

Que se passe-t-il si un client m'envoie un message en dehors des heures de travail ?▼
Le bot répond automatiquement 24h/24, 7j/7 — même la nuit, les weekends et les jours fériés. Vos clients ne seront jamais sans réponse.
Est-ce que je peux modifier mes produits après l'inscription ?▼
Oui, depuis votre dashboard vous pouvez ajouter, modifier ou supprimer des produits à tout moment. Les changements sont pris en compte immédiatement par le bot.
Puis-je utiliser mon numéro personnel pour le bot ?▼
<
Comment le bot sait-il quoi répondre ?▼
Le bot utilise Claude, une intelligence artificielle d'Anthropic. Il connaît votre catalogue, votre description de boutique et votre personnalité. Plus vous remplissez votre profil avec de détails, meilleures seront les réponses.
Que se passe-t-il quand mon essai gratuit expire ?▼
Vous recevrez un message WhatsApp 3 jours avant l'expiration, puis la veille. Le jour de l'expiration, votre dashboard vous proposera de choisir un plan. Le bot sera suspendu jusqu'au renouvellement de votre abonnement via My Touchpoint.
Mon ID de boutique est-il sécurisé ?▼
Votre ID est un identifiant unique de 36 caractères aléatoires — il est pratiquement impossible à deviner. Conservez-le précieusement et ne le partagez avec personne.
Comment fonctionne le bot WhatsApp ?▼
Le bot répond automatiquement aux messages WhatsApp de vos clients 24h/24. Il présente votre catalogue, prend les commandes et vous notifie à chaque nouvelle vente. Pas de configuration technique requise !
