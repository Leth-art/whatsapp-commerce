Politique de confidentialité

Dernière mise à jour : 1er avril 2026 WaziBot SaaS — Afrique de l'Ouest

WaziBot ("nous", "notre", "la plateforme") est un service SaaS qui fournit des assistants commerciaux intelligents via WhatsApp aux commerçants d'Afrique de l'Ouest. Cette politique explique comment nous collectons, utilisons, stockons et protégeons vos données personnelles ainsi que celles de vos clients.

🔒

Notre engagement envers vous

Nous ne vendons jamais vos données à des tiers. Vos données vous appartiennent. Nous les utilisons uniquement pour faire fonctionner et améliorer votre boutique WaziBot.

1 Qui sommes-nous

WaziBot est une plateforme SaaS hébergée sur Render (serveurs cloud) avec une base de données PostgreSQL sécurisée. Notre service est accessible via :

  • Site web : wazibot.com
  • Dashboard marchand : interface web accessible depuis tout navigateur
  • Notifications WhatsApp : alertes de nouvelles commandes envoyées au commerçant via l'API officielle Meta

2 Données collectées

Pour les commerçants :

  • Informations boutique : nom, ville, pays, devise, description
  • Coordonnées : adresse email, numéro WhatsApp
  • Identifiants techniques : identifiant boutique unique, paramètres de notification WhatsApp (stockés chiffrés, jamais exposés)
  • Paiements traités par My Touchpoint (Mobile Money) — aucune donnée bancaire stockée
  • Plan d'abonnement et date d'expiration

Pour les clients de votre boutique :

  • Numéro WhatsApp (identifiant anonyme fourni par Meta)
  • Nom fourni volontairement dans la conversation
  • Historique des commandes et conversations avec le bot
  • Date de dernière interaction (pour les relances automatiques)

3 Utilisation des données

  • Faire fonctionner l'assistant IA intégré à votre boutique en ligne
  • Traiter et suivre les commandes de vos clients
  • Envoyer des relances automatiques aux clients inactifs (plans Pro et Business)
  • Générer vos rapports hebdomadaires et statistiques dashboard
  • Vous notifier des rappels d'abonnement et renouvellements
  • Améliorer les performances de l'IA et du service

4 Partage des données

Nous partageons uniquement les données strictement nécessaires avec ces partenaires :

  • Anthropic (Claude AI) — génération des réponses du bot de votre boutique (questions produits, prix, livraison). Seul le contenu du message est transmis, sans données personnellement identifiables du client
  • Meta / WhatsApp Business API — envoi et réception des messages via l'infrastructure officielle
  • My Touchpoint — traitement sécurisé des paiements Mobile Money
  • Render — infrastructure serveur et base de données PostgreSQL sécurisée
⚠️

Votre responsabilité envers vos clients

En utilisant WaziBot, vous êtes responsable d'informer vos clients que votre boutique utilise un assistant IA pour répondre aux questions et prendre les commandes. Nous vous recommandons d'en faire mention sur votre page boutique.

5 Conservation des données

  • Données boutique et clients : conservées tant que votre compte est actif
  • Historique des commandes : 24 mois après la dernière commande
  • Conversations IA : 90 jours glissants
  • Après suppression de compte : toutes les données effacées sous 30 jours

6 Sécurité

  • Chiffrement en transit sur toutes les communications (HTTPS/TLS)
  • Base de données PostgreSQL avec accès restreint et sauvegardes automatiques
  • Tokens WhatsApp stockés de manière chiffrée, jamais exposés publiquement
  • Routes API protégées par authentification par clé sur tous les endpoints
  • Aucun stockage de mots de passe — authentification par ID unique sécurisé

7 Dashboard et site boutique

  • Dashboard marchand : gérez vos commandes, produits et paramètres via /merchant
  • Mini-site boutique : vitrine publique générée automatiquement sur /boutique/votre-slug
  • Seul votre ID boutique est stocké localement dans votre navigateur (localStorage)
  • Toutes les communications sont chiffrées (HTTPS)

8 Vos droits

  • Accéder à vos données via votre dashboard WaziBot
  • Modifier vos informations dans les paramètres
  • Exporter vos commandes et données clients en contactant notre support
  • Supprimer votre compte et toutes vos données sur demande
  • Contester tout traitement de données vous concernant

9 Cookies et tracking

  • localStorage uniquement pour mémoriser votre ID boutique (connexion automatique)
  • Aucun cookie tiers, aucun pixel tracking, aucune publicité
  • WaziBot est un espace sans publicité — nous ne permettons pas aux annonceurs de cibler nos utilisateurs

10 Modifications

Nous pouvons mettre à jour cette politique à tout moment. En cas de changement important, vous serez notifié par WhatsApp ou email. La date de dernière mise à jour est toujours indiquée en haut de cette page.
